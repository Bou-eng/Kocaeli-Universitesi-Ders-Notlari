Durum çizeneði yanda görülen sonlu özdevinirin tanýdýðý kümeyi bir düzgün deyim olarak bulalým. Makinenin A, B ve C adlý üç durumu bulunduðu için üç deðiþkenli aþaðýdaki denklem sistemi kurulur:

A = + A0 + B1_____________________ (1)
B = A0 + B1 + C0__________________ (2)
C = B0 + C1 _________________________(3)

 

Üçüncü denkleme Teorem 2.3 uygulandýðýnda:

C = B01* __________________________________________________________(4)

elde edilir.

Ýkinci denklemde C’nin yerine B01* deðeri konularak da:

B = A0 + B1 + B01*0 = A0 + B(1 + 01*0) _____________________________(5)

elde edilir.

Teorem 2.3 kullanýlarak yukarýdaki denklem çözüldüðünde ise:

B = A0(1 + 01*0)* _________________________________________________(6)

elde edilir.

Birinci deklemde B’nin yerine yukarýdaki deðer konulduðunda ise birinci denklem aþaðýdaki biçime dönüþür:

A = + A0 + A0(1 + 01*0)*1 = + A[0 + 0(1 + 01*0)*1] ______________(7)

Teorem 2.3 kullanýlarak yukarýdaki denklem çözüldüðünde ise A’ya karþý gelen aþaðýdaki düzgün deyim elde edilir:

A = [0 + 0(1 + 01*0)*1]*= [0 + 0(1 + 01*0)*1]*_____________________(8)

Denklem (6)’da A’nýn yerine (8)’de bulunan deðeri konularak B’ye karþý gelen aþaðýdaki düzgün deyim elde edilir.

B = [0 + 0(1 + 01*0)*1]* 0(1 + 01*0)*________________________________(9)

Son olarak da denklem (4)’de B’nin yerine (9)’da bulunan deðeri konularak C’ye karþý gelen düzgün deyim elde edilir.

C = [0 + 0(1 + 01*0)*1]* 0(1 + 01*0)* 01*

Denklem sistemi çözülüp, her bir duruma karþý gelen düzgün deyim elde edildikten sonra, makinenin tanýdýðý kümeye karþý gelen düzgün deyim aþaðýdaki gibi elde edilir.

T(M2.2) = B + C = B + B01* = B( + 01*)
